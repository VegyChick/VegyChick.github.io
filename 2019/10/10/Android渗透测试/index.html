adb shell

反编译

第一步：把你的apk后缀名改为zip，然后解压后，就会发现有个classes.dex，这里面就是java源码了。

第二步：我们需要吧dex文件转为jar格式的文件，用到了一个工具叫dex2jar( 下载链接)，我们把它下载来后解压(下载来后是zip文件)。

第三步：我解压后是放在F:\Android\apktools\dex2jar-2.0目录下，现在把dex文件拷贝到该目录下。

第四步：cmd窗口进入F:\Android\apktools\dex2jar-2.0，然后输入命令：d2j-dex2jar.bat classes.dex回车

完成后我们在F:\Android\apktools\dex2jar-2.0目录下就会多了个classes-dex2jar.jar文件，这个就是我们需要的jar文件，接下来就是如何查看java代码了，借用一个工具jd-gui，下载链接，下载完解压，然后用它打开classes-dex2jar.jar就可以了，如果你的apk经过混淆，那么看到的都是a、b之类的。

Apktool

反编译：java -jar apktool.jar d -f TmmsSuite.apk -o yhzj

打包：apktool b test -o yhtest.apk

生成密钥：keytool -genkeypair -v -keystore xample.keystore -alias test -keyalg RSA -keysize 2048 -validity 10000

出现乱码：chcp 936

签名：jarsigner -keystore xample.keystore -digestalg SHA1 -sigalg SHA1withRSA zytest.apk test
